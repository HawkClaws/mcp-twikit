# MCP-Twikit

A Model Context Protocol (MCP) server for Twitter search.

## Installation

```json

"twitter": {
    "command": "uvx",
    "args": ["--from", "git+https://github.com/adhikasp/mcp-twikit", "mcp-twikit"],
    "env": {
    "TWITTER_USERNAME": "@example",
    "TWITTER_EMAIL": "me@example.com",
    "TWITTER_PASSWORD": "secret",
    }
}
```
